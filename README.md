# Cartelera se ha mudado

Este repositorio era la versión 100% navegador de Cartelera (sin cuenta,
`localStorage`, sin sincronizar entre dispositivos). Ya no se mantiene.

Cartelera vive ahora en **https://cartelera-app.onrender.com** — backend
Spring Boot multiusuario, login con Google, biblioteca sincronizada entre
dispositivos. El código está en
[`cartelera-app`](https://github.com/kcuevasb/cartelera-app).

`index.html` en este repo es solo una redirección automática a esa URL,
para que los enlaces/accesos directos viejos de GitHub Pages no queden
rotos.
